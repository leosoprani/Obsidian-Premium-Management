# Obsidian Premium Management 💎

Sistema de gestão de elite para locações e hospitalidade, com foco em estética premium, automação e experiência de usuário de alto nível.

## 🚀 Visão Geral

O **Obsidian** é uma plataforma completa que une um **Portal Administrativo** robusto a um **App Mobile** exclusivo para proprietários. O sistema utiliza uma identidade visual moderna baseada em *Glassmorphism* e *Mesh Gradients* para entregar uma experiência fluida e futurista.

---

## 📱 Mobile App (Owner Portal)

Desenvolvido para entregar o controle total dos imóveis na palma da mão.

### ✨ Principais Funcionalidades
- **Dashboard Dinâmico**: Visualização de métricas mensais, reservas ativas e status financeiro.
- **Gestão de Hóspedes**: Formulários rápidos para cadastro de novos visitantes.
- **Calendário Inteligente**: Sincronização e visualização de disponibilidade.
- **Chat Real-time**: Comunicação direta e fluida integrada ao backend.
- **Design de Elite**: Layout otimizado para iOS e Android, com efeitos de transparência (Glassmorphism) e centralização cirúrgica.

### 🛠️ Especificações Técnicas (Mobile)
- **Framework**: React Native + Expo SDK 54.
- **UI/UX**: Estilo Glassmorphism customizado, otimizado para performance Android (hardware-acceleration friendly).
- **Localização**: 100% adaptado para o padrão brasileiro (`DD/MM/AAAA`).

---

## 🖥️ Portal Administrativo (Web)

O "motor" da operação, focado em controle operacional e financeiro.

### ⚙️ Funcionalidades Admin
- **Gestão Operacional**: Cronogramas de Limpeza, Manutenção e Reparos.
- **Controle de Bloqueios**: Indisponibilidade rápida de unidades.
- **Gestão Financeira**: Relatórios de faturamento e despesas.
- **Monitoramento**: Logs de sistema e auditoria de usuários.

## 🛠️ Configuração e Instalação

### Pré-requisitos
- Node.js (v18+)
- Expo CLI
- Git

### Servidor (Backend)
1. Navegue até a raiz do projeto.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor:
   ```bash
   npm start
   ```

### Mobile (Mobile/)
1. Navegue até a pasta `mobile/`.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o ambiente Expo:
   ```bash
   npx expo start
   ```
